現場別ソリューション

どんな現場でも、
「繋がらない」をなくします。

🎪 催事・祭事

マラソン / 花火大会 / 地域イベント

広大な会場でも、スタッフ全員がリアルタイムで繋がる

単発イベントには購入よりレンタルが経済的。規模・スタッフ構成に合わせてIP無線と簡易無線を組み合わせた最適構成をご提案します。

簡易無線（登録局）× 一般スタッフ全員
IP無線 × 本部・救急・警備など基幹スタッフ
→ レンタルで必要な期間だけ利用可能
🎸 ライブ・フェス

音楽フェス / コンサート / スポーツイベント

常設は購入機で、大規模時はレンタルで追加増台

通常規模は購入済み無線機を使用し、突発的な大規模イベント時にはレンタルで柔軟に対応。IP無線で離れた舞台袖と制作本部を繋ぎます。

簡易無線（登録局）× 通常スタッフ（購入）
IP無線 × 制作本部・基幹スタッフ
→ 大規模時の増台はレンタルで対応
🏬 店舗・施設

百貨店 / ショッピングモール / レジャー施設

普段の接客連携から、万が一の防災対応まで一台で

日常業務の連絡効率を上げつつ、災害時には同じ端末が防災連絡網に切り替わります。大丸福岡天神店でも採用実績があります。

登録局 or 特定小電力 × 各フロアスタッフ
IP無線 × 本部・各拠点・防災センター
→ 日常業務＋防災の二役を担う
🏗️ 工事現場

建設 / 土木 / クレーン作業 / インフラ工事

クレーンオペレーターと地上の合図、確実に届ける

クレーン操作など安全に直結する現場では通信の確実性が最優先。用途・規模に応じた機種を選定し、MCA無線からの移行も全面サポートします。

同時通話無線 × クレーン操作（2〜10名）
登録局 or IP無線 × 一般作業員・管理者
→ MCA無線からの移行も対応
🚨 防災・緊急対応

自治体 / 企業BCPチーム / 防災訓練 / NGO

地上インフラが壊れても通信を止めない3層構成

業務無線・LTE・Starlinkを組み合わせた冗長構成で、どんな災害時も通信を維持。台風19号の支援活動でも実績のある構成です。

IP無線（WAVE PTX）× 基幹スタッフ
Starlink × 完全バックアップ
→ 防災訓練の立会い・年次サポートも対応
⚠️ 特殊環境

石油 / 化学プラント / ガス設備 / 防爆地帯

防爆仕様の無線機で、危険区域でも安全に通信

引火・爆発リスクのある環境では、一般の無線機は使用できません。国内防爆検定合格済みの専用機種で、安全基準を満たした通信環境を構築します。

防爆無線機 PD798Ex × 危険区域作業員
本質安全防爆構造 / 国内検定合格品
→ 法規制への適合もサポート

機種の選び方

迷ったら、この2つの質問で
決まります。

通信エリアで選ぶ

どのくらいの距離・範囲で使いますか？

  • 同じ建物・フロア・敷地内（数百m以内）
  • 免許の手続きをせずすぐ使いたい
  • コストをできるだけ抑えたい
→ 簡易無線（登録局）がおすすめ
免許不要で当日から使用可。コストパフォーマンス最高。

通信エリアで選ぶ

広いエリアや複数拠点を繋ぎたいですか？

  • 山間部・離島・屋外広域での使用
  • 複数拠点・離れた場所を一つのグループに
  • GPS追跡・ビデオ連携も必要
→ IP無線 / WAVE PTXがおすすめ
全国どこでも繋がる。GPS・ビデオ機能も統合。

用途で選ぶ

短期の単発利用ですか？継続利用ですか？

  • 年数回のイベント・工事など一時的な利用
  • 機材管理・保守の手間を省きたい
  • 初期投資を抑えたい
→ レンタルがおすすめ
必要な期間だけ、最高のコンディションで利用可能。

用途で選ぶ

常時使用・防災対応が必要ですか？

  • 毎日の業務で継続的に使用する
  • いざというときに絶対に動く体制が必要
  • 複数台を長期間・安定的に運用したい
→ 月額サブスク（Resilience SaaS）がおすすめ
購入・保守・訓練をまとめた月額プランで安心運用。
